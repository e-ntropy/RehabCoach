    • This app adopts deep learning models from the mmpose framework to enable efficient and accurate real-time 2D human pose estimation. By simply using the phone camera, it can intelligently analyze key body joints and evaluate body posture.

    The models and algorithms are optimized for medical rehabilitation scenarios, significantly reducing model computation for low-latency real-time pose detection and analysis. It also features custom pose matching algorithms designed for common rehabilitation motions, comparing users' live poses against standard poses to provide corresponding training feedback and improvement suggestions for intelligent personalized guidance.

    Compared to traditional rule-based methods, this app utilizes deep learning techniques for more precise and smarter rehabilitation assistance. It reduces reliance on professional therapists - users can get training guidance anytime through the mobile app without location constraints, making it more convenient and efficient. It currently supports various common rehabilitation motions, and more will be continuously added to enrich training options for better user experience.

    In summary, this app fully exploits computer vision and AI to creatively apply human pose estimation to medical rehabilitation, aiming to provide an economical and effective solution for personalized smart rehabilitation.
